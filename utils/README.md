# E-commerce Test Automation Framework

## Overview
This project is a sample automation framework built using:

- Selenium (UI automation)
- Pytest (test execution)
- Requests (API testing)
- SQLite (database validation)

It demonstrates real-world testing across multiple layers:
UI → API → Database

---

## Features

- Page Object Model (POM)
- Data-driven testing using pytest
- API validation
- Database validation
- Integration testing (UI + API + DB)

---

## Project Structure

tests/
- ui_tests → UI test cases
- api_tests → API test cases
- integration_tests → end-to-end validation

pages/
- Page classes for UI automation

utils/
- Database helper functions

data/
- Test data and mappings
